# Paladin's Quest
A browser-based adventure game built with vanilla JavaScript and HTML — play as a paladin navigating challenges and enemies on a quest to victory.
> ⚠️ **Game Link** [Paladin's Quest](https://ayubmo-2.github.io/Paladins-Quest/).
---
## Tech stack
* **Language** — JavaScript (93.7%), HTML (6.3%)
* **Rendering** — HTML Canvas / DOM
* **Build** — no build step, runs in the browser
---
## Features
* **Paladin character** — player-controlled hero with movement and combat
* **Enemy encounters** — face enemies and obstacles throughout the quest
* **Level progression** — advance through stages to complete the game
* **Browser-based** — no installation required, runs entirely in the browser
